## 🚀 Featured Projects

### [Model The Game](https://modelthegame.com) · [Repo](https://github.com/ameya-pandit/nfl-predictor)
End-to-end NFL game prediction system. Siamese neural network trained on 5 years of play-by-play data, achieving **65% accuracy across 272 matchups in the 2025 season**. Full MLOps pipeline: Docker-containerized services, FastAPI backend, PostgreSQL, and a live React dashboard.

**Stack:** Python · PyTorch · FastAPI · PostgreSQL · Docker · React · Tailwind · Railway · Vercel

### [Cadence](https://github.com/ameya-pandit/habit-tracker)
AI-powered habit tracker with **voice-driven check-ins via OpenAI Whisper**, a 5-tier habit classification engine, and LLM-generated weekly digests via GPT-4o-mini.

**Stack:** React · Node.js · Express · MongoDB · OpenAI API · Tailwind
